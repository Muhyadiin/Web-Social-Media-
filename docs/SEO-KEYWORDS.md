# SEO Strategy & Keywords

## Goal

Rank for high-intent searches from developers, contractors, architects and owners
looking to **price, design or de-risk** a construction project — and convert that
traffic into quote requests.

## On-page SEO (already implemented)

- ✅ Unique title tag + meta description per page
- ✅ One `<h1>` per page, logical `<h2>`/`<h3>` hierarchy
- ✅ Descriptive, keyword-aware headings & copy
- ✅ Canonical URLs
- ✅ Open Graph + Twitter Card tags
- ✅ JSON-LD structured data (ProfessionalService, Person, Service, ContactPage)
- ✅ `sitemap.xml` + `robots.txt`
- ✅ Mobile-first, fast-loading, semantic markup
- ✅ Internal linking (services ↔ home ↔ contact)

## Off-page & technical SEO checklist

- [ ] Set the real domain everywhere (canonical, OG, sitemap, robots)
- [ ] Verify the site in **Google Search Console**; submit `sitemap.xml`
- [ ] Create / claim **Google Business Profile** (local SEO)
- [ ] Add real `alt` text to all images
- [ ] Build a consistent **LinkedIn** presence and link to the site
- [ ] Earn backlinks: guest posts, directory listings, industry associations
- [ ] Publish blog articles consistently (target the keywords above)
- [ ] Add Google Analytics 4 / Plausible + conversion tracking
- [ ] Keep page speed high (compress images, lazy-load)

## Measuring success

Track in Search Console & analytics:
- Impressions & clicks for target keywords
- Average position for primary keywords
- Organic sessions → quote-form conversions
- Top landing pages and queries
