# 30-Day Content Calendar

A ready-to-run, 30-day launch plan built on the 7 content pillars and the 5-3-2
mix (educational / proof / promotional). Each day has a **primary platform** (post
natively) and should be **repurposed** to the others.

**Legend — Pillars:** `STR` Structural Engineering · `QS` Quantity Surveying ·
`COST` Construction Costs · `BOQ` BOQ Tips · `SITE` Site Engineering ·
`PM` Project Management · `BRAND` Personal Branding
**Type:** `E` Educational · `P` Proof/Inspiration · `R` Promotional

> Suggested daily slot: post **08:00–09:00** local (before the workday) or
> **19:00–20:00**. LinkedIn favours weekday mornings.

## Repurposing workflow (per piece)

```
1 idea  →  1 YouTube or LinkedIn long-form
        →  1 carousel (LinkedIn + Instagram)
        →  1 Reel / TikTok / Short (vertical clip)
        →  1 Facebook post
        →  1–2 Stories (poll / teaser)
```

## Pre-launch checklist

- [ ] Optimize all profiles (bio, photo, banner, link)
- [ ] Batch-create Week 1 content in advance
- [ ] Prepare 5–10 reusable carousel/Reel templates in Canva/CapCut
- [ ] Set up a scheduler (Meta Business Suite / Buffer / Metricool)
- [ ] Decide your DM keyword ("QUOTE") and a quick reply script
- [ ] Add UTM links so you can see which posts drive website leads
