# Website assets

Upload image files here (exact filenames matter — the pages reference them).

| File | Used on | Notes |
| --- | --- | --- |
| `portrait.png` | Home hero + About page | Your business portrait (background-removed export). Shown in a rounded card. |
| `project1.png` … `project6.png` | Portfolio + Home preview | Project images. If a file is missing, the page falls back to a gradient label automatically. |
| `og-image.png` *(optional)* | Social link previews | 1200×630 share image. |
| `favicon.ico` *(optional)* | Browser tab | |

**How to upload (GitHub web UI):** open this folder on GitHub → **Add file → Upload files** →
drag your images in (named exactly as above) → **Commit changes** to `main`. The
site redeploys automatically and the images appear.
